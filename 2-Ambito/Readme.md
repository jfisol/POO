# Ámbito

El ámbito es la región del programa donde es conocido el método o variable, los datos seran reconocidas en determinada seccion del programa

# Ámbito Local
Declarado dentro del mismo bloque de códio pudiendo ser funciones o clases.


# Ámbito a nivel de Objeto
 Todos los métodos que pertenezcan a esa clase pueden utilizar las variable.


# Objeto this
Es el ambito de objeto, se hace uso cuando se referencia a la variable que esta declarada en la clase. `this.numero1`
